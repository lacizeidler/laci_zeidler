Tim the Tool Man

Front End Capstone

Tim the Tool Man application allows a contractor to keep track and communicate with all his or her clientele. The manager is able to see all new, in progress, and finished projects and filter through each type with ease. There is access to each customer's individual profile with all their requests. The manager is able to delete requests and edit the status of the request. The customer is able to see their own requests and edit the request description if needed. Both parties are able to communicate with each other about specifc requests through the message thread within the customer profile. The contractor never has to worry about a request from a customer or communication through multiple platforms.
